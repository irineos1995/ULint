                      Directives - table of contents
                  
    
                  
    BootstrapVue directives and directive group plugins
    
                  
    
                    
                    
                  
    
                  
    Popover —
                      Add BootstrapVue popovers to any element on your site,
                        using Bootstrap v4 CSS for styling and animations. Popovers are tooltips on
                        steroids.Scrollspy —
                      Automatically activate BootstrapVue navigation or list
                        group components based on scroll position to indicate which link is currently active in the
                        viewport.Tooltip —
                      Add custom BootstrapVue tooltips to any element.
                        Tooltips can be triggered by hovering, focusing, or clicking an element.
